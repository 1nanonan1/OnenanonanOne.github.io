# OnenanonanOne.github.io
weather application


The placeholder is opened for the end user to type in their desired city. The city's temperature is pulled from the API  getWeather() and pushed to the tag - rendered on the interface.

Some features can include the humidity and wind speeds. 
Future goals, models can be utilized for an agent to read out whether a jacket, sweater or umbrella may be needed in certain range of the day.

#Steps
 1.Server: if not configured, get all requirements, mentioned from lecture topics in downloading Express, etc.
else, run this -> node server.js

2. Open Browser, localhost:3000
3. Weather Cast App is launched.
